# Popper

> Popper 借助[Popper.js](https://popper.js.org/)实现，主要作为其他组件的辅助类使用，尽量不要单独使用。由 Popper 派生的组件主要有下面几个：

- Tooltip: 用于文字提示
- Popover: 用于轻量级的 Modal
- Select: 选择器
- Dropdown: 下拉菜单
- Menu: 菜单
- Cascader: 级联选择器
- DatePicker: 日期选择器
- TimePicker: 时间选择器
